# 索菲斯：瓦尼拉的崛起 [WIP]
這是個生存工藝角色扮演模組。玩家扮演被遺棄的英雄，從他方世界穿越到瑟頓領域的瓦尼拉世界。瓦尼拉因神而殞落，而英雄將帶領他們再次崛起，並獲取更為強大的力量。玩家將運用強大的工藝、科技系統，從無到有地創造出強大的實力，並透過這些科技持續強化自我，使命運來臨時，我們將打破世界的瓶頸，再次崛起。
